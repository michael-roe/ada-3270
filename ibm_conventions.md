# IBM User Interface Conventions

## Function Keys

* PF1 - Help
* PF3 - Exit
* PF7 - Prev
* PF8 - Next

# Scroll Indicator

When a logical page contains more than one physical page, this can be
indicated by either:

"More:" in the top right, followed by "+" if there are subsequent pages
and "-" if there are previous pages 

or "Line nn to nn of nn".

# Command Prompt

If a screen has an input field for a command prompt, it is labelled
"Command ===>"
